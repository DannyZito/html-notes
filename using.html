Basic Codeing Instructions

You can let web browsers know that you are using HTML by starting your document with a document type declaration. The declaration looks like this: < !DOCTYPE html> must be the first line of code in all of your HTML documents. To create HTML structure and content, we must add opening and closing < html> tags, like so: < !DOCTYPE html> < html> < / html> Anything between the opening < html> and closing tags will be interpreted as HTML code. A html tag goes at the beginning and end of an < html> CODE, a < p> tag goes at the beginning and end of a PARAGRAPH. Closing tags have a forward slash (/) inside of them, directly after the left angle bracket. The opening and closing head tags <head> </head> typically appear as the first item after your first HTML tag. The title can be specified directly inside of the (< head>) element, by using a <title> tag. Whitespace makes the code easier to read, example:

IMAGES

The <img> tag allows you to add an image to a web page. This is another example of a self-closing tag. src must be the uniform resource locator (URL) of the image. Example: < img src="image-location.com" /> Visually impaired users often browse the web with the aid of screen reading software. When you include the alt attribute, the screen reading software can read the image's description out loud to the visually impaired user. ADDING LINKS

You can add links to a web page by adding an anchor element <a> and including the text of the link in between the opening and closing tags.

You must also add a href attribute like so;

<a href="link to site"> description of link</a> For a link to open in a new window, the target attribute requires a value of _blank. The target attribute can be added directly to the opening tag of the anchor element, just like the href attribute. <a href="https://en.wikipedia.org/wiki/Brown_bear" target="_blank">The Brown Bear</a>

A relative path is a filename that shows the path to a local file (a file on the same website, such as ./index.html) versus an absolute path (a full url, like www.codecademy.com/learn/ruby which is stored in a different folder). The ./ in ./index.html tells the browser to look for the file in the current folder. <a href="./contact.html">Contact</a> Linking to elements on the same page or to other pages on the same site is called navigation. HTML has a special tag called <nav> that is used to wrap these links in order to organize the content on your web page.

Before displaying data, you must first create the table that will contain the data by using the <table> element. <table> </table> The first step in entering data into the table is to add rows using the table row element: <tr>. Rows aren't sufficient to add data to a table. Each cell element must also be defined. In HTML, you can add data using the table data element: <td>Number</td> Table data doesn't make much sense without titles to describe what the data represents. To add titles to rows and columns, you can use the table heading element: <th>. Note, also, the use of the scope attribute, which can take one of two values:

row - this value makes it clear that the heading is for a row. col - this value makes it clear that the heading is for a column. for example: <th scope="row">Name</th>
