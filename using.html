Basic Codeing Instructions

You can let web browsers know that you are using HTML by starting your document with a document type declaration. The declaration looks like this: < !DOCTYPE html> must be the first line of code in all of your HTML documents. To create HTML structure and content, we must add opening and closing < html> tags, like so: < !DOCTYPE html> < html> < / html> Anything between the opening < html> and closing tags will be interpreted as HTML code. A html tag goes at the beginning and end of an < html> CODE, a < p> tag goes at the beginning and end of a PARAGRAPH. Closing tags have a forward slash (/) inside of them, directly after the left angle bracket. The opening and closing head tags <head> </head> typically appear as the first item after your first HTML tag. The title can be specified directly inside of the (< head>) element, by using a <title> tag. Whitespace makes the code easier to read, example:

Comments begin with < !-- and end with -->. Any characters in between will be ignored by your browser. < h1> - used for main headings. All other smaller headings are used for subheadings. Example: <h2>, <h3>, <h4>. The <em> tag will generally render as italic emphasis. The <strong> will generally render as bold emphasis. Ordered lists are like unordered lists, except that each list item is numbered. You can create the ordered list with the <ol> tag and then add individual list items to the list using <li> tags.

IMAGES

The <img> tag allows you to add an image to a web page. This is another example of a self-closing tag. src must be the uniform resource locator (URL) of the image. Example: < img src="image-location.com" /> Visually impaired users often browse the web with the aid of screen reading software. When you include the alt attribute, the screen reading software can read the image's description out loud to the visually impaired user.
